Disk Cleaner

Safe viewer for large files and caches

This UI does not delete files. It only previews storage usage and exports a cleanup plan. Apply the plan with the shell script using Trash by default.

How to use
  1. Run a scan to produce a report JSON: 
    ./disk_cleaner.sh --scan-output /tmp/disk_cleaner_report.json --min-size 50M --older-than 30 --downloads
  2. Click “Load Scan Report” and choose the JSON.
  3. Filter and select items, then click “Export Cleanup Plan”.
  4. Apply the plan (dry-run first): 
    ./disk_cleaner.sh --apply-from /tmp/disk_cleaner_plan.json --dry-run
./disk_cleaner.sh --apply-from /tmp/disk_cleaner_plan.json --apply --trash --yes
